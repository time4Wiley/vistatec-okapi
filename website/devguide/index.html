Glossary Guide Content Okapi Framework

The Okapi Framework Developer's Guide provides you with some help on how to use the framework's components in your own programs and scripts, as well as how to develop your own Okapi-compatible components so they can be easily shared among the different Okapi tools.

IMPORTANT: Parts of this developer's guide is not completely up-to-date with the latest version of the framework. You should refer to the Java documentation for the most up-to-date information.
