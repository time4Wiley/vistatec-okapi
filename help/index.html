Okapi Framework - Help

Topics

The main help system of the Okapi Tools is located on a public wiki. It includes help for the following parts:

  • Tikal (Command-line tool)
  • Rainbow (Localization toolbox)
    • The utilities provided with Rainbow
  • Ratel (Segmentation rules editor)
  • CheckMate (Quality checker)
  • Longhorn (Batch processing server)
  • The extraction filters
  • The processing steps and Rainbow's pre-defined pipelines
  • The TM and MT connectors

For example on how to use the tools, see the tutorials.

For more information on how to use the filters in your own scripts and applications: see the online Developer's Guide.

For help, questions and discussion: see the users group.

For the source code: see the project repository.
