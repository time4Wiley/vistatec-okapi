Overview

This step imports the text units of an input document into a Pensieve TM.

Takes: Filter events
Sends: Filter events

Each text unit is imported into the specified TM. If the text unit is segmented, each segment is imported.

If the text unit does not have text for the specified target locale, that text unit is not imported. In other words.

In a text unit segmented, if the number of segments is different between the source and the target, the whole text unit is imported instead of the segments.

Parameters

Directory of the TM where to import -- Enter the directory of the TM where to import the text unit. If the TM does not exist it will be created. If the TM exists already the entries will be added to the existing ones. You can use the variable ${rootDir} in the path.
