Overview

This step merges back an extracted content coming as filter evenst into its original format.

Takes: Filter events
Sends: Raw document

This step is used to take filter events and merge them back into their original file format.

To convert a raw document to filter events use the Raw Document to Filter Events step.

Parameters

This step has no user parameters. All the parameters are provided by the calling application.
