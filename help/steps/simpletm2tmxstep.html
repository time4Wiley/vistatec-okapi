Overview

This step generates a TMX document for a given SimpleTM input.

Takes: Raw Document
Sends: Raw Document

Each input document is processed and the step generates a corresponding TMX document (which is sent as the Raw Document for the next step). The name of the TMX file is the name of the input document plus a .tmx extension.

Note: The SimpleTM format is obsolete, and will be supported only for a time for backward compatibility. Use the Pensieve TM for your TMs.

Parameters

This step has no user parameters. All the parameters are provided by the calling application.
