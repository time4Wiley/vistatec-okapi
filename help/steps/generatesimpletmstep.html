Overview

This step generates a SimpleTM translation memory from a set of bilingual documents.

Takes: Filter events
Sends: Filter events

This step creates a single SimpleTM translation memory from a set of bilingual documents, usually TMX files.

Note: The SimpleTM format is obsolete, and will be supported only for a time for backward compatibility. Use the Pensieve TM for your TMs.

If the input is a TMX file the property Txt::GroupName and the property Txt::FileName are imported.

Parameters

Path of the TM -- Enter the full path of the SimpleTM translation memory to generate. If the TM exists already it will be overwritten. You can use the variable ${rootDir} in the path.
