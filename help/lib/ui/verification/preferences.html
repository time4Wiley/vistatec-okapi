CheckMate

User Preferences

If you are using an Okapi Tool after the M9 release, you should be using the wiki online help:
http://okapiframework.org/wiki/index.php?title=CheckMate

This dialog box allows you to edit the user-specific options for CheckkMate.

Edit Fields

These options determine how the text is displayed in the source and target edit fields.

Source edit field

Font -- Click this button to select a different font, font size or style for the source edit field.

Text Color -- Click this button to change the color of the source text.

Background -- Click this button to change the color of the background of the source edit field.

Right to left script -- Set this option to enabled the Right-To-Left processing of the source text. Use this option for the languages that use scripts such as Arabic, Hebrew, Thaana, etc.

Target edit field

Font -- Click this button to select a different font, font size or style for the target edit field.

Text Color -- Click this button to change the color of the target text.

Background -- Click this button to change the color of the background of the target edit field.

Right to left script -- Set this option to enabled the Right-To-Left processing of the target text. Use this option for the languages that use scripts such as Arabic, Hebrew, Thaana, etc.
