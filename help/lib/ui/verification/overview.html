CheckMate

Overview

If you are using an Okapi Tool after the M9 release, you should be using the wiki online help:
http://okapiframework.org/wiki/index.php?title=CheckMate

CheckMate is a Java application that perform various quality checks on bilingual/multilingual translated documents.

CheckMate and the all the components of the Okapi Framework are free and open-source.The application runs on top of Java (version 1.6 and higher) and runs on any platform where SWT is supported, including the Windows, Linux and Macintosh systems.
