File

New Session

Creates a new empty session. If you have a modified un-saved session already opened, the tool will ask you if you want to save your changes before.

Shortcut: Ctrl+N

Open

Opens an existing session. If you have a modified un-saved session already opened, the tool will ask you if you want to save your changes before.

Shortcut: Ctrl+O

Save Session

Saves the current session. If the session has no name, the tool will prompt you to choose one.

Shortcut: Ctrl+S

Save Session As

Saves the current session under a new name.

Recent Session Files

Opens the session selected in the list of most recently used sessions. If the current session has been modified but not saved, you will be asked if you want to save the changes.

Clear Recent Session Files List

Removes all sessions from the list of most recently used sessions.

Add Document

Adds a document to the list of documents to check in this session.

Shortcut: Ctrl+Insert

You can also add a document by:

  • dropping the file onto the issue list of the main window.
  • opening the Session Settings dialog box and use the Add button there.

Note: All documents in the session must be for the same language pair. All documents must be bilingual/multilingual translated documents (for example: XLIFF, PO, TMX, TS, PO, etc).

Session Settings

Opens the Session Settings dialog box, where you can set the documents to process and other options.

User Preferences

Opens the User Preferences dialog box, where you can set user-specific options such as the fonts used to display the source and target text.

Exit

Exits the editor. If you have a modified un-saved document already opened, the tool will ask you if you want to save your changes before.

Shortcut: Alt+F4

Issues

Reset All Disabled Issues

Resets to enabled all issues currently set as disabled, on all documents.

Edit Configuration

Opens the Quality Check Configuration dialog box, where you specify the types of checks to perform.

Load Configuration

Loads an existing configuration file.

Save Configuration

Save the current configuration into a file.

Check All

Performs all the verifications currently selected for all the documents currently in the session.

Shortcut: Shift+F5

Check Current Document

Performs all the verifications currently selected for only the document of the issue currently selected.

Shortcut: F5

Generate Report

Generates a report of all issues (disabled issues are not included in the report). The path of the report is specified in the Quality Check Options dialog box.

Help

Help Topics

Opens the main help page for this application.

Shortcut: F1

How To Use CheckMate

Opens the help page to get you started with this application.

Check For Updates

Checks if there is an updated version of CheckMate that is available. If CheckMate is invoked from a parent application, this command checks for updates of the parent application.

This command connects to Internet and send only the name of the product and its version number.

Send Feedback

Sends an email with feedback to the development team.

Submit Bug Report

Opens the page where you can submit a bug report.

Submit Feature Request

Opens the page where you can submit a feature request.

Users Group

Opens the Okapi Tools users group (http://tech.groups.yahoo.com/group/okapitools/) where you can discuss Okapi-related topics with other users.

About CheckMate

Opens the About CheckMate dialog box where you can find version and system information.
