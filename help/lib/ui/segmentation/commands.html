File

New

Creates a new empty SRX document in the editor. If you have a modified un-saved document already opened, the tool will ask you if you want to save your changes before.

Shortcut: Ctrl+N

New With Simple Default Rules

Creates a new SRX document in the editor with simple default rules. Use this command as a quick way to start a new document, or to test a few rules. If you have a modified un-saved document already opened, the tool will ask you if you want to save your changes before.

Open

Opens an existing SRX document in the editor. If you have a modified un-saved document already opened, the tool will ask you if you want to save your changes before. You can also open an SRX document by dragging and dropping the file anywhere in the editor's main window.

Shortcut: Ctrl+O

Load From Clipboard

Loads an SRX document from the Clipboard. If you have a modified un-saved document already opened, the tool will ask you if you want to save your changes before.

The Clipboard must contain a plain text copy of a valid SRX document.

Save

Saves the current document. If the document has no name, the tool will prompt you to choose one. The tool will prompt you to confirm the save if the document was loaded as an SRX version 1.0 file. All documents are saved to SRX version 2.0,

Shortcut: Ctrl+S

Save As

Saves the current document under a new name.

Copy To Clipboard

Copies the current document to the Clipboard. Note that the SRX document copied to the Clipboard does not include Okapi extensions elements.

Exit

Exits the editor. If you have a modified un-saved document already opened, the tool will ask you if you want to save your changes before.

Shortcut: Alt+F4

Tools

Test Segmentation on a File

Takes an input file and segment it with the current SRX document and settings, and generate an output file with the segments marked up. This opens the Test Segmentation on a File dialog box where you can find more details on the requirements for the input document.

Help

Help Topics

Opens the main help page for this application.

Shortcut: F1

How To Use Ratel

Opens the help page to get you started with this application.

Check For Updates

Checks if there is an updated version of Ratel that is available. If Ratel is invoked from a parent application, this command checks for updates of the parent application.

This command connects to Internet and send only the name of the product and its version number.

Send Feedback

Sends an email with feedback to the development team.

Submit Bug Report

Opens the page where you can submit a bug report.

Submit Feature Request

Opens the page where you can submit a feature request.

Users Group

Opens the Okapi Tools users group (http://tech.groups.yahoo.com/group/okapitools/) where you can discuss Okapi-related topics with other users.

SRX 2.0 Specification

Opens the SRX 2.0 Specification online page.

About Ratel

Opens the About Ratel dialog box where you can find version and system information.
