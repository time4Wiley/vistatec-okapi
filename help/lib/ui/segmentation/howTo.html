How To Use Ratel

If you are using an Okapi Tool after the M9 release, you should be using the wiki online help:
http://okapiframework.org/wiki/index.php?title=Ratel

If you want to modify an SRX document:

  1. Select Open in the File menu, and select the document to open.
  2. Select the group of rules to modify in the Language rules currently displayed list box.
  3. Use the Edit button to edit a selected rule, or the Add button to create a new one.
  4. You can arrange the order of the rules with the Move Up and Move Down buttons.
  5. Select Save in the File menu to save your modifications.

You can also use the Clipboard to transfer the full set of rules back and forth between an external application and the editor.
