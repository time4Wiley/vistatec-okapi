Rainbow

Overview

If you are using an Okapi Tool after the M9 release, you should be using the wiki online help:
http://okapiframework.org/wiki/index.php?title=Rainbow

Rainbow is an application that that offers a palette of utilities aiming to help you during the process of localizing documentation, on-line help and software-related material.

The utilities and filters Rainbow utilizes are independent plug-in components of the Okapi Framework and Rainbow is merely a graphical interface to launch them. Note that you can also use Rainbow as a command-line tool. Here are some of the things you can do with Rainbow:

  • Convert a file from one character set encoding to another.
  • Prepare files in different formats to be translated.
  • Compare two translations of the same source text.
  • Remove or add the Byte-Order-Mark in UTF-8 files.
  • Search and replace some text only in the translatable part of a file.
  • Pseudo-translate files.
  • ...and much more. See the Utilities section for more information.

Rainbow and the all the components of the Okapi Framework are free and open-source.The application runs on top of Java (version 1.6 and higher) and runs on any platform where SWT is supported, including the Windows, Linux and Macintosh systems.
