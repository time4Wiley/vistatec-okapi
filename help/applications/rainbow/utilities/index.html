Utilities Okapi Framework

If you are using an Okapi Tool after the M9 release, you should be using the wiki online help:
http://okapiframework.org/wiki/index.php?title=Steps

Rainbow is distributed with several pre-installed utilities:

  • Translation Package Creation
  • Translation Package Post-Process
  • ID-Based Alignment
  • Encoding Conversion
  • Byte-Order-Mark Conversion
  • RTF Conversion

You can also use pre-defined pipeline, or create you own to use the Okapi steps components.
