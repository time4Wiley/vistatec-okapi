Rainbow

Input Document Properties

If you are using an Okapi Tool after the M9 release, you should be using the wiki online help:
http://okapiframework.org/wiki/index.php?title=Rainbow

This dialog box allows you to modify the properties associated with one or more input documents.

If more than one document is selected, the values for the last selected document are shown. The values you then choose are applied to all the selected documents.

Filter configuration

Top drop-down list -- Select the filter to associate with the input document, or choose <None> to associate no filter with the input document.

Second list box -- Select the configuration associated with the input document, the default one is the first one listed.

For more information about the filters and configurations available, see the Filters section.

View / Edit -- Click this button to view or edit the parameters file of the configuration currently selected. If no editor is available for this type of parameters, the parameters file is opened in a default text editor. Pre-defined configurations are available for viewing only. Custom configurations can be edited.

Create --Click this button to create a new configuration, based on the one currently selected. You will be prompted for its name.

Delete -- Click this button to delete the configuration currently selected. Note that you cannot delete pre-defined configurations.

More -- Click this button to open the Filter Configurations dialog box, where you can manage all the configurations available for the project.

Encodings

Source encoding -- Enter the name of the encoding to use for the source document. Leave this entry empty to use the default source encoding that is defined in the Languages and Encodings tab of the main window.

Target encoding -- Enter the name of the encoding to use for the target document. Leave this entry empty to use the default target encoding that is defined in the Languages and Encodings tab of the main window.
